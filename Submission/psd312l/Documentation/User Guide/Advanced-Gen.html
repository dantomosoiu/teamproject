Advanced Settings - General Tab

Save Settings on Exit

Save any changes to settings as they happen and again on exit.

Hide Manual Camera Controls

Hides the camera control panel

Show co-ordinates

Show co-ordinates for each point on the NavMesh. Currently unimplemented due to an inability to save BitmapText in JMonkey.

Show Origin

Show the origin of the scene graph

Print Debug Info to Command Prompt

If running in a command prompt or terminal assorted information will be printed.

Person Base Speed

Decides the base speed of the population. WARNING - At time of writing setting this to 0 is permitted.

Confirm to Exit

Displays a confirm dialogue when exiting.

Show Frames Per Second

Displays frames per second in the corner of the JMonkey Canvas. Useful for performance testing on lower spec machines.

Show Routes

Allows turning off of displaying of routes. This would allow minor performance increases. Currently not implemented.

NavMesh Color

If you don't like white choose something else!
