========================
An evacuation simulator based on the JMonkey Engine. 

Designed and developed by:
Michael Kilian; 
Tony Lau; 
Dan Tomosoiu; 
Hector Grebbell; 
Peeranat Fupongsiripan.

March 2013, University of Glasgow.
========================
To run the project from the command line, go to the dist folder and
type the following:

java -jar "TallShipEvacuationSimulator.jar" 
========================
Platform dependent (Windows, Linux, MacOSX) zip archives are also provided to ease distribution.
