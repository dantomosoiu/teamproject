Menu Bar

  1. File Menu
    Import Model - See Import Model (Currently Not Implemented)
    Exit - Closes the simulator
  2. Settings Menu
    Advanced Settings - Opens the Advanced Settings panel. See Advanced Settings sections for more.
    Save Settings - Saves the current settings
    Import Settings - Imports settings from a .data file.
    Export Settings - Exports current settings to a .data file.
    Restore Factory Settings - Resets the settings to default. This will also reload the original Glenlee model if another has been imported.
  3. Help Menu
    About - Opens a small about pane.
    Help - Opens the Help Dialogue.

JME Canvas

The Large display to the upper left is the graphical representation of the evacuation. As standard this will show a white wireframe representation of the navigatable surface of the ship. Spread accross this will be the population. These are represented by figurines.
The camera angle can be changed either with keyboard controls or the panel in the bottom right (see camera controls).
Addtional options such as display of frames per second are available via the Advanced Settings Panel.

Sidepanel (Top)

Population size - The number of people to populate the ship with

Time Elapsed - The time the current evacuation has been running for.

Population Remaining - The number of people remaining on the ship.

Safely Evacuated - The number of people who have safely left the ship.

View Controls

Show Wireframe - A toggle to show the navigation mesh as a wireframe

Show Decks - A toggle to show the physical representation of the floorplan of the ship.

Show Far-Side Hull - A toggle to show the far side of the hull.

Camera Location - Allows "snapping" to predetermined camera locations
