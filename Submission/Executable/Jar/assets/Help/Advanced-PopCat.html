Advanced Settings - Population Categories Tab

Allows custom setting of categories of people and their attributes. To be heavily expanded! Currently implementation relating to these variables is limited.

Categories can be added with the Add button. The remove button removes the currentlly selected category.

Min Speed - Minimum speed a person in this category can move at.

Max Speed - Minimum speed a person in this category can move at.

Stress values - Currently not used

Color - Color to show these people in. Currently implementation is tempramental at best.
